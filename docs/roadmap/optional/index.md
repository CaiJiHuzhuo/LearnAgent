## 选修建议

**如果你的项目需要数据安全**：优先学习 O.1（本地模型）+ O.7（企业认证）+ O.8（沙箱执行）

**如果你的项目需要处理图像**：优先学习 O.3（多模态）

**如果你需要分析数据**：优先学习 O.5（SQL Agent）

**如果你在做开发工具**：优先学习 O.6（代码 Agent）+ O.8（沙箱执行）

**如果你需要提升模型专业性**：优先学习 O.2（微调）
